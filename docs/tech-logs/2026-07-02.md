## 已确认判断

- MVP 目标是个人练手 OJ 平台，但可以支持小规模课堂或公网试用。
- 后端使用 C++ 和 `cpp-httplib`。
- 前端使用原生 HTML/CSS/JavaScript，由 C++ 服务托管静态文件。
- 判题模式只做 ACM 标准输入输出。
- 提交语言只支持 C++。
- 数据库使用 MySQL 8.0。
- 判题队列使用 MySQL 任务表，不只放在内存里。
- 判题沙箱使用 Docker；MVP 允许后端直接调用 `docker run`。
- 部署形态是一台 Linux 机器同时运行后端、MySQL 和 Docker 判题环境。
- UI 风格参考 `https://datacollector.cn/`：白色、克制、工具型、弱装饰。
- 测试策略包括 Google Test、LLM + curl、Python requests、Playwright。
